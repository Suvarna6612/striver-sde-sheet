# Day 1 - Arrays

## 1. Set Matrix Zeroes

Problem:
Given a matrix, if an element is 0, set its entire row and column to 0.

Technique Used:

* Marker Technique
* In-place Matrix Manipulation
* Space Optimization

Complexity:

* Time: O(N × M)
* Space: O(1)

---

## 2. Pascal's Triangle

Problem:
Generate Pascal’s Triangle for the given number of rows.

Technique Used:

* Pattern Building
* Dynamic Programming style relation
* Combinatorics Observation

Complexity:

* Time: O(N²)
* Space: O(N²)

---

## 3. Next Permutation

Problem:
Find the next lexicographically greater permutation of the array.

Technique Used:

* Greedy Algorithm
* Lexicographical Ordering
* Array Manipulation

Complexity:

* Time: O(N)
* Space: O(1)

## 2. Maximum Subarray (Kadane’s Algorithm)
